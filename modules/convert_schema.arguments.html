convert_record_fields (fields, root_name)
Convert each field of an avro-schema record to a scalar graphql type or an input object.

Parameters:

  • fields table list of fields of the avro-schema record fields format
  • root_name string topmost part of namespace

Returns:

    table args -- map with type names as keys and graphql types as values

convert (avro_schema, opts)
Convert avro-schema type to GraphQL scalar or InputObject.

Parameters:

  • avro_schema (table or string) avro-schema with expanded references
  • opts table

    the following options:

    • type_name (string; optional) name for GraphQL type instead of one from avro-schema full name (considered for record / record*)

    • context (table) avro-schema processing context:

    • field_name (string; optional) name of currently parsed field

    • path (table) point where we are in avro-schema

Returns:

    table GraphQL scalar or InputObject
