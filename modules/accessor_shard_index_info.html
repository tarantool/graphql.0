get_index_info (collection_name, index_name)
Get index object from net_box under the shard module.

The function performs some optimistic consistency checks and raises an error in the case. It caches results and returns a result from the cache for succeeding calls.

XXX: Implement some cache clean up strategy and a way to manual cache purge.

Parameters:

  • collection_name string
  • index_name string

Returns:

    index object
