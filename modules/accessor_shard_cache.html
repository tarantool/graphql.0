cache_fetch (self, batches)
Fetch data to the cache.

Parameters:

  • self table accessorshardcache instance
  • batches table

    requests batches in the following format:

     batches = {
     [field_name] = {
         collection_name = <string>,
         index_name = <string or nil>,
         keys = <...>,
         iterator_opts = <...>,
     },
     ...
 }

Returns:

    table

    the following structure or nil:

     {
     fetch_id = <number>, -- identifier of the fetched data
     stat = {             -- data to update statistics
         fetches_cnt = <number>,
         fetched_tuples_cnt = <number>,
         full_scan_cnt = <number>,
         index_lookup_cnt = <number>,
     }
 }
